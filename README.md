## 📌 About the Project

This project is a **Figma design to HTML/CSS conversion** for a digital agency website. The goal was to faithfully recreate the designer's vision in code while ensuring the site looks great and functions well across all device sizes — from mobile phones to large desktop screens.

Every section — from the hero banner to the leadership team — has been carefully structured with semantic HTML and styled with modern CSS techniques to match the original Figma mockup as closely as possible.
